Double click on the exe

The first time you run it, you'll be asked to choose a username and an optional key.

Menu ---

[1] Create server
[2] Join server
[3] Exit

Create server ---

You'll need to download and set up playit.gg (or any similar tunneling tool) and create a UDP tunnel for your server's port. Copy the address it gives you and paste it into SpyEyeUser/public_address — or just enter it when the program asks for a public address when you create a server. Make sure the tunnel protocol is set to UDP, not TCP, or connections won't work.

Enter a port (press Enter for the default 50007).
Optionally set a password — anyone joining will need it.


Join server ---


Pick a saved server from the list, or choose the last option to add a new one (name, IP, port).
Enter the server's password if it has one.
